# 📚 ScholarChain  
## Decentralized Achievement Credential System  

ScholarChain is a **blockchain-based credential platform** that transforms achievements into secure, verifiable digital credentials.  

Instead of scattered certificates and documents, achievements are represented as **[Soulbound Tokens](ca://s?q=Explain_Soulbound_tokens)** — unique, non-transferable assets that can be independently verified through blockchain technology.  

---

## 🚀 Features  

- 🎖 **Soulbound Achievement Tokens** — Non-transferable proof of achievements  
- 🔍 **Credential Verification System** — Decentralized validation of records  
- 🌐 **Public Verification Portal** — Recruiters & institutions can verify instantly  
- ⛓ **[Ethereum Sepolia](ca://s?q=Ethereum_Sepolia_testnet)** Integration — Smart contract deployment  
- 🔑 **[MetaMask](ca://s?q=What_is_MetaMask_wallet)** Authentication — Secure wallet-based ownership  
- 📦 **[Pinata + IPFS](ca://s?q=IPFS_Pinata_storage)** — Decentralized metadata storage  
- 🖥 **React Dashboard** — User-friendly credential management
