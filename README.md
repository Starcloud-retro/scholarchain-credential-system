\# ScholarChain



\## Decentralized Achievement Credential System



ScholarChain is a blockchain-based credential platform that transforms achievements into secure, verifiable digital credentials.



Instead of relying on scattered certificates and documents, achievements are represented as Soulbound Credential Tokens that can be independently verified through blockchain technology.



\---



\## Features



\- Soulbound Achievement Tokens

\- Credential Verification System

\- Public Verification Portal

\- Ethereum Sepolia Integration

\- MetaMask Authentication

\- Pinata IPFS Metadata Storage

\- React Dashboard

\## Architecture



Issuer

↓

Issues Credential

↓

Metadata Uploaded To Pinata

↓

CID Generated

↓

Soulbound Token Minted

↓

Stored On Ethereum

↓

Visible In Dashboard

↓

Publicly Verifiable
